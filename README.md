# ProjetoDIO

O ProjetoDIO é um projeto em ASP.NET CORE que contém uma aplicação principal em MVC e uma API.
  
 ##### Tela aplicação MVC 
![Tela MVC](/assts/imgs/TelaMVC.png)
##### Tela API/Swagger
![Swagger](/assts/imgs/TelaSwagger.png)

## Tecnologias
- .NET
- Migrations
- Entity Framework Core
- Visual Studio 2019
